## e-Govなどの通知はPDFにしてほしい
マイクロ法人を始めてからというもの、xmlファイルを受信する機会がとても増えました。上記の設定をしておけば簡単にxmlファイルを開くことができますが、PCを持っていない人も多いのでPDFファイルで通知してもらえたらとても気が利くのになぁ…と思ってしまいます。（PDF通知にするとなにか不具合があるのかもしれませんが）
